Rules for constructing Primary constants

Integer

A number that represents an integral value that cannot be changed is known as integer constant. An "integer constant" may have a decimal (base 10), octal (base 8), or hexadecimal (base 16).

Rules for constructing Integer Constants

* Integer constant can be positive as well as negative.

* Integer without any sign is considered as positive integer.

* It must not have a decimal point.

* Commas or blanks are not allowed within an integer constant.

* An integer constant must not exceed the range -32768 to 32767.

* An integer constant must have at least one digit.

Example of integer constants

* 3

* +5

* -432

Real Constant

Real constants are often called Floating Point constants. It is a digit sequence containing a decimal point or exponent or both.


The real constants could be written in two forms__Fractional from and Exponential from.

Rules for constructing Real Constants

following rules must be observed while constructing real constants expressed in fractional form:

* Real constant can be positive as well as negative.

* Real constant without any sign is considered as positive integer.

* It must have a decimal point.

* Commas or blanks are not allowed within a real constant.

* A real constant must not exceed the range -3.4e38 to 3.48e38

* A real constant must have at least one digit.

Example of Real Constants

* +3.5

* 5.0

* -35.67

Character Constant

A character is a single alphabet, a single digit or a single special symbol enclosed within single inverted commas.

Rules for constructing Character Constants

* Both the inverted commas should point to the left for a character.

Example, 'C' is a valid character constant whereas 'C' is invalid.

* The maximum length of a character constant can be 1 character.

Example of Character Constants

* 'c'

* 'm'

* '25'

* '$'

Q.2. What is variable ? Explain in detail.

C Variable

An entity that may change during program execution is called a variable. Variable names are names given to locations in memory.

C variables can be of following types:

variable Declaration

* In declaration of a variable, no space is reserved for it.

* A variable can be declared several times

Syntax

data_type identifie1, identifier2,.......identifier_n;

Example

int i,j;

Variable Defination

* In defining a variable, its space gets reserved for it in memory.

* A variable can be defined only once.

Rules for constructing Variable Names

* Length of variable names should be 1 to 31 characters.

* Digit must not be taken as first character of a variable name.

* Commas and blanks are not allowed within a variable name.

* only underscore can be used in a variable name as a special symbol.

LITERALS:

LITERALS: In computer science, a literal is a notation for representing a fixed value in source code. Almost all programming languages have notations for atomic values such as integers, floating-point numbers, strings, and booleans; some also have notations for elements of enumerated types and compound values such as arrays, records, and objects.


In constrast to literals, variables or constants are symbols that can take on one of a class of fixed values, the constant being constrained not to change. Litrals are often used to initialize variables, for example:

int a=1;

String s="cat";


In some object-oriented languages (like ECMAScript), objects can also be represented by literals. Methods of this object can be specified in the object literal using function literals. The brace notation below, which is also used for array literals, is typical for object literals:

{name:"cat",length:57}
